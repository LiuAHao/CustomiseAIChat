# 阶段 0 测试验收清单

适用范围：平台骨架阶段

---

## 1. 工程级检查

- [ ] `scripts/test.ps1` 可运行
- [ ] `scripts/build.ps1` 可运行
- [ ] `go test ./...` 全部通过
- [ ] API 二进制可生成
- [ ] Worker 二进制可生成

---

## 2. 服务级检查

- [ ] API 可启动
- [ ] `/healthz` 返回 200
- [ ] 默认管理员可登录
- [ ] 登录后返回 token

---

## 3. 主链路检查

- [ ] 可读取当前用户信息
- [ ] 可读取默认工作区
- [ ] 可创建 Agent
- [ ] 可创建 Agent 版本
- [ ] 可发布版本
- [ ] 可创建 Tool
- [ ] 可发起 Task
- [ ] 高风险 Tool 可进入待审批状态
- [ ] 审批通过后任务完成

---

## 4. 前端联调检查

- [ ] 前端可启动
- [ ] 首页可打开
- [ ] 默认管理员可登录
- [ ] 后端 token 可返回并显示

---

## 5. 当前阶段通过标准

满足以下条件可视为阶段 0 完成：

1. Go 测试通过
2. 后端服务可启动
3. 登录链路可用
4. Agent / Tool / Task / Approval 主链路打通
5. 前端可完成基本登录联调
