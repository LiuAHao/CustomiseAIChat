# 前端组件模块说明

`web/console/src/components` 负责控制台的页面骨架、导航和共享组件。

## 当前组件职责

- `Sidebar.tsx`
  负责左侧导航、Agent 列表、收起 / 展开交互、设置入口与退出登录入口。
- `AgentDashboard.tsx`
  负责主工作区视图切换，包含：
  - 新建 Agent
  - Agent 聊天页
  - 技能与应用
  - MCP 配置
  - 模型配置
  - 系统设置
- `AuthPage.tsx`
  负责登录页。
- `AppIcon.tsx`
  负责统一导航图标与页面顶部图标的图标源，避免同一功能使用多套图标。

## 当前前端设计原则

- 不暴露“工作区”概念
- 页面数据优先来自真实后端接口，不在前端伪造资源列表
- `Skill / MCP` 统一分为“平台资源”和“我的资源”
- 点击左侧 Agent 进入该 Agent 的聊天页，而不是跳回新建页
- 聊天页采用主白底布局，底部输入框常驻，尽量减少无意义留白

## 关键交互约束

- 左侧栏支持收起与展开
- 收起后仅保留图标，不展示无 Agent 时的空提示文字
- 聊天页消息区应尽量占满主内容区，输入框固定在底部
- 所有页面保持同一套浅色、圆角、低噪声视觉语言
